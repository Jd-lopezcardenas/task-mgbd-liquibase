# Investigación sobre Liquibase

## ¿Qué es Liquibase?
Liquibase es una herramienta que permite gestionar cambios en bases de datos mediante archivos versionados llamados changelogs.

## ¿Qué es un changelog?
Es un archivo que contiene todos los cambios que se aplicarán a la base de datos.

## ¿Qué es un changeset?
Es una unidad de cambio dentro de un changelog. Cada changeset representa una modificación específica.

## ¿Qué es rollback?
Es la capacidad de deshacer cambios aplicados previamente en la base de datos.

## Importancia
Permite:
- Control de versiones
- Trabajo en equipo
- Reproducibilidad
- Automatización de despliegues
